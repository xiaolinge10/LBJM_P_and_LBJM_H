# LBJM_P_and_LBJM_H
This is Source code for simple implements of LBJM_P and LBJM_H algorithms

# Data_of_JW_PA_BK
This is the data of estimating the ground-state energy of the hydrogen molecular under Jordan-Wigner (JW), parity (PA) and Bravyi-Kitaev (BK) mappings.

# Data_of_density_matrix
This is the data of reconstructed density matrix, where Sheet1 and Sheet2 are real and imaginary parts of the experimental quantum state.

# Data_of_experiment_results
This is the data of estimating a Hamiltonian composed of 50 Pauli strings and a cluster Hamiltonian using experimental data.
